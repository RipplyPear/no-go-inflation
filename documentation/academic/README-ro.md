## Despre proiect

**No-go Inflation** este un joc 2D multiplayer cooperativ în care jucătorii dezvoltă economii locale, construiesc clădiri de producție, colectează resurse și tranzacționează pe o piață comună. Obiectivul colectiv este creșterea economică fără inflație excesivă.

Proiect realizat ca lucrare de licență la Academia de Studii Economice din București, specializarea Informatică Economică.
